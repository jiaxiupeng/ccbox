# 更新日志

本文件记录 CCBox 各版本的变更。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

---

## [1.0.2] - 2026-07-07

### 新增
- **权限模式选择**：设置页新增「权限模式」区块，支持 Claude Code 的 6 种权限模式（默认 / 自动接受编辑 / 计划 / 自动 / 不再询问 / 跳过授权）
- 写入 `~/.claude/settings.json` 的 `permissions.defaultMode`，安全合并，保留用户已有的 allow/deny/ask 列表
- 用户选择持久化到 CCBox 偏好设置，切换供应商不会丢失
- 危险模式（bypassPermissions）用红色警告样式区分

### 修复
- **检查更新无反馈**：修复启动时检查失败后，手动检查仍走缓存返回"已是最新"的问题；现在失败后必定重新发起请求

---

## [1.0.1] - 2026-07-07

### 新增
- **应用内自动更新**：基于 Tauri 2 官方 updater 插件，启动后自动检查新版本
- **更新徽章提示**：发现新版本时，标题栏显示带红点动画的「新版本」徽章
- **更新对话框**：显示版本号、更新内容、下载进度条，一键下载安装并重启
- **设置页检查入口**：关于区可手动触发「检查更新」
- **签名校验**：每个版本经 Ed25519 密钥签名，客户端下载后自动验签，防篡改
- **GitHub Actions 发版流水线**：打 tag 即自动构建、签名、发布

### 优化
- **README 重写**：专业徽章行 + 居中下载指引区 + Star History 折线图
- **项目历史整理**：压缩为单条干净的初始提交
